==> run_time    --> This parameter controls the duration of the animation.

==> grid_xstep  --> This specifies the horizontal spacing between grid lines, defining the width of each cell in the grid.

==> grid_ystep  --> This specifies the vertical spacing between grid lines, defining the height of each cell in the grid.
